La versión completa con CI/CD pipeline, Docker multi-stage, Makefile inteligente, bootstrap de herramientas y configuraciones de editor profesionales — todo adaptado al tipo de tu proyecto.


💎 ¿Qué incluye este pack premium?

🔥 spark:blast — CI/CD + Docker
Genera un pipeline de GitHub Actions con linting y tests para tu tipo de proyecto. Incluye Dockerfile multi-stage optimizado (python:slim, node:alpine, rust:slim, nginx:alpine para web) y .dockerignore profesional.

🔧 spark:forge — Makefile Inteligente
Crea un Makefile con 10 targets: help, build, test, clean, lint, run, install, docker-build, docker-run. Cada target adaptado al lenguaje de tu proyecto. El comando 'help' documenta todos los targets automáticamente.

🌍 spark:env — Bootstrap de Herramientas
Detecta qué tools tienes instaladas (python3, node, cargo, g++, shellcheck...) y cuáles te faltan. Te muestra el comando exacto para instalarlas y, si quieres, las instala por ti.

🧰 spark:kit — Templates de Editor
Genera .editorconfig (para todo el equipo), más la configuración específica de tu lenguaje: ruff.toml para Python, .prettierrc + .eslintrc.json para TypeScript, .clang-format para C++, rustfmt.toml para Rust.
